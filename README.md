## 開発環境
* python3系

* pillow


## コマンドライン引数
`python flower_generater.py --color 126 0 255 --number 6 --type 0`  
or  
`python flower_generater.py -c 126 0 255 -n 6 -t 0`

### パラメータ
* 花弁の色 (RGBが0~255)

* 花弁の数 (0~12)

* 花弁の種類 (0-6)
